# ERP管理统计报表功能测试文档

## 1. 测试概述

### 1.1 测试目的
验证ERP管理统计报表页面的各项功能是否正常工作，确保数据统计准确、图表展示正确、页面交互流畅。

### 1.2 测试范围
- 统计概览数据展示
- 采购/销售趋势图表
- 库存预警列表
- 热销商品排行榜
- 订单状态分布饼图
- 供应商/客户统计

### 1.3 测试环境
- 前端地址：http://localhost:3000/erp
- 后端地址：http://localhost:8001
- 测试账号：admin / admin123
