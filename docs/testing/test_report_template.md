## 1. 测试概述

### 1.1 测试目标
本次测试旨在验证 ERP 进销存系统的各项功能是否符合需求规格，确保系统的正确性、稳定性和安全性。

### 1.2 测试范围
- 采购订单管理：创建、审核、入库、取消、删除
- 销售订单管理：创建、审核、出库、取消、删除
- 库存管理：查询、调整、变动记录
- 产品管理：CRUD 操作
- 供应商管理：CRUD 操作
- 客户管理：CRUD 操作
- 系统管理：登录、权限验证

### 1.3 测试方法
- 单元测试：Go Testing
- 集成测试：Postman
- 端到端测试：Cypress
