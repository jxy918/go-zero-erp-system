# 库存预警与多计量单位功能测试方案

## 7. 测试风险与注意事项

1. **数据库字段迁移**：确保products表已添加min_stock, max_stock, safety_stock字段
2. **数据一致性**：测试前确保测试数据已清理或隔离
3. **权限验证**：确保测试用户有相关操作权限
4. **接口认证**：测试接口时携带有效的JWT token
