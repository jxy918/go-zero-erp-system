# ERP管理统计报表功能需求文档

## 1. 需求概述

### 1.1 功能背景
为企业管理者提供一个直观的ERP数据统计报表页面，展示采购、销售、库存等核心业务数据，支持数据可视化分析和决策支持。

### 1.2 功能目标
- 展示企业运营核心数据指标（采购金额、销售金额、库存总量、订单总数）
- 提供采购/销售趋势分析图表
- 显示库存预警信息
- 展示热销商品排行榜
- 统计订单状态分布

### 1.3 目标用户
- 企业管理员
- 运营管理人员

### 3.1 性能要求

## 4. 数据模型

### 4.1 统计概览数据结构

| 字段名 | 类型 | 说明 |
|--------|------|------|
| purchaseAmount | float | 本月采购金额 |
| salesAmount | float | 本月销售金额 |
| totalInventory | int | 当前库存总量 |
| totalOrders | int | 本月订单总数 |

### 4.2 趋势数据结构

| 字段名 | 类型 | 说明 |
|--------|------|------|
| dates | array[string] | 日期列表 |
| purchaseData | array[float] | 采购金额数据 |
| salesData | array[float] | 销售金额数据 |

### 4.3 库存预警数据结构

| 字段名 | 类型 | 说明 |
|--------|------|------|
| productName | string | 商品名称 |
| quantity | int | 当前库存 |
| threshold | int | 预警阈值 |

### 4.4 热销商品数据结构

| 字段名 | 类型 | 说明 |
|--------|------|------|
| rank | int | 排名 |
| productName | string | 商品名称 |
| quantity | int | 销售数量 |
| amount | float | 销售额 |

### 4.5 订单状态分布数据结构

| 字段名 | 类型 | 说明 |
|--------|------|------|
| pending | int | 待审核数量 |
| approved | int | 已审核数量 |
| completed | int | 已完成数量 |
| cancelled | int | 已取消数量 |

## 5. API接口设计

### 5.1 获取统计概览数据

| 属性 | 值 |
|------|------|
| 方法 | GET |
| 路径 | /erp/statistics/overview |
| 认证 | 需要 |

**响应示例**：
```json
{
  "code": 0,
  "data": {
    "purchaseAmount": 1250000.00,
    "salesAmount": 980000.00,
    "totalInventory": 5680,
    "totalOrders": 156
  },
  "message": "success"
}
```

### 5.2 获取趋势数据

| 属性 | 值 |
|------|------|
| 方法 | GET |
| 路径 | /erp/statistics/trend |
| 参数 | days (可选，默认30) |
| 认证 | 需要 |

**响应示例**：
```json
{
  "code": 0,
  "data": {
    "dates": ["2024-01-01", "2024-01-02", ...],
    "purchaseData": [12000, 15000, ...],
    "salesData": [8000, 12000, ...]
  },
  "message": "success"
}
```

### 5.3 获取库存预警数据

| 属性 | 值 |
|------|------|
| 方法 | GET |
| 路径 | /erp/statistics/inventory-alert |
| 认证 | 需要 |

**响应示例**：
```json
{
  "code": 0,
  "data": [
    {"productName": "商品A", "quantity": 5, "threshold": 10},
    {"productName": "商品B", "quantity": 3, "threshold": 15}
  ],
  "message": "success"
}
```

### 5.4 获取热销商品排行

| 属性 | 值 |
|------|------|
| 方法 | GET |
| 路径 | /erp/statistics/top-products |
| 参数 | limit (可选，默认10) |
| 认证 | 需要 |

**响应示例**：
```json
{
  "code": 0,
  "data": [
    {"rank": 1, "productName": "商品X", "quantity": 120, "amount": 24000},
    {"rank": 2, "productName": "商品Y", "quantity": 98, "amount": 19600}
  ],
  "message": "success"
}
```

### 5.5 获取订单状态分布

| 属性 | 值 |
|------|------|
| 方法 | GET |
| 路径 | /erp/statistics/order-status |
| 认证 | 需要 |

**响应示例**：
```json

  "data": {
    "pending": 25,
    "approved": 30,
    "completed": 85,
    "cancelled": 16
  },
  "message": "success"
```
